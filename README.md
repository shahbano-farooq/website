# Shahbano Farooq — Portfolio Website

A personal portfolio showcasing HCI and visualization research, teaching experience, and interactive D3.js projects.

## Features

- **Home** — Interactive D3.js life timeline visualization spanning education, industry, teaching, and research
- **Projects** — Bird Migration visualization and M.Sc. thesis (PairedVis)
- **Teaching** — D3.js dashboard of course history with bar chart and semester heatmap
- **Publications** — Research output from CV
- **CV** — Full curriculum vitae with PDF download

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Customization

Edit `lib/data.ts` to update biography, life events, publications, teaching records, and project descriptions.

Place your profile photo at `public/profile.jpg` and update the home page hero section.

## Tech Stack

- Next.js 15 (App Router)
- React 19
- TypeScript
- Tailwind CSS 4
- D3.js 7
